## 📖 Project Overview

GestureSphere Explorer is an AI-powered Virtual Reality project that uses Computer Vision and Machine Learning to recognize hand gestures in real time. Users can interact with a virtual environment without any physical controller, making the experience more natural and immersive.

The system detects hand landmarks using MediaPipe, processes gesture data using TensorFlow, and translates recognized gestures into actions inside a VR environment.
